# Developer Test

This application is an API for an ad quality management team to sort ads based on a number of specific features.

## Instructions
- Install Docker https://www.docker.com/get-started
- `make build` to build the containers
- `make start` to start the containers
- `make stop` to stop the containers
- `make restart` to restart the containers
- `make prepare` to install dependencies with composer (once the project has been created)
- `make run` to start a web server listening on port 1000 (8000 in the container)
- `make logs` to see application logs
- `make ssh-be` to SSH into the application container
